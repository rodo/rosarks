Rosarks

Rosarks API

Rosarks is a simple API to find amenities located near a point, results are available in JSON format. Datas come from OpenStreetMap project.

/bicycle_rental/

Where you can take a bicycle from a bicycle rental/sharing system, like Veli'b in Paris (France) (sample)

/bus_stop/

Where you take or leave a public bus (sample)

/subway_station/

Subway stations with line information like colors (sample)

/tramway_station/

Tramways stop/station with line information like name and colors (sample)
