Rosarks

HOWTO use Rosarks

Using rosarks is easy as doing an HTTP GET

  • Urls are in form "GET /amenity_code/longitude/latitude/"
  • longitude and latitude are in EPSG:4326 if you use another projection you have to convert before.

Available amenities

All available amenities are described on homepage be sure we'll work hard to add new one if you request it.

Precision

You can add a parameter on url to change the default precision, this parameter must be a float between 100 and 2000. It's units is meters, the default precision is 1000. A query with precision will looks like /amenity/lon/lat/50.0/ with a precision of 50 meters.

We reduce the search on a precision of 5 digits, ie the queries around lon,lat=2.123456789,42.0 and lon,lat=2.12345,42.0 will return the same results
